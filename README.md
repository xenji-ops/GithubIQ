# 🧠 GitHubIQ

![GitHubIQ Banner](https://via.placeholder.com/1200x400/000000/FFFFFF?text=GitHubIQ+-+Engineering+Intelligence)

**GitHubIQ** is a premium, SaaS-grade GitHub profile analyzer designed for software engineers, engineering leaders, and technical recruiters. It goes far beyond simple star counts to generate deep, actionable insights into repository quality, contribution patterns, language intelligence, and career readiness.

---

## ✨ Features

- **Developer Intelligence Score:** A multi-factor assessment grading profiles on consistency, repo quality, project diversity, and documentation.
- **Language Intelligence:** Parses public repos to analyze stack mastery, bytes written, and language diversity over time.
- **Repository Quality Audits:** Scores repos on README completeness, licensing, community signals (stars, forks), and maintenance activity.
- **Contribution Rhythms:** Identifies developer archetypes through streak analysis, commit cadence, and active days.
- **Career Matching:** Maps engineering skills to roles with confidence scores and structured growth paths.
- **Sleek, Premium UI:** Built with dark-mode first glassmorphism, fluid micro-animations, and responsive SaaS design principles.

## 🛠️ Getting Started

To get a local copy up and running, follow these simple steps:

### Prerequisites

- Node.js 18.0 or later
- npm, yarn, pnpm, or bun

### Installation

1. Clone the repo
   ```bash
   git clone https://github.com/xenji-ops/GithubIQ.git
   ```
2. Navigate to the project directory
   ```bash
   cd GithubIQ
   ```
3. Install dependencies
   ```bash
   npm install
   ```
4. Start the development server
   ```bash
   npm run dev
   ```
5. Open [http://localhost:3000](http://localhost:3000) in your browser.
